# saeb_microdados_dashboard
Projeto de análise, tratamento e visualização interativa dos microdados do SAEB (Sistema de Avaliação da Educação Básica), abrangendo informações de alunos, docentes, diretores e escolas desde 1995-2023. O objetivo é transformar dados brutos em dashboards interativos que apoiem a compreensão da educação pública brasileira ao longo do tempo.
